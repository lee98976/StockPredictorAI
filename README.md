# StockPredictorAI
An AI for long-term trading that predicts the close price 15 days in the future. Once you run main.py, a discord bot will activate. 

### Commands:
```bash
$Stock [Stock Symbol]
```

## Python Version:
3.12.4

## Libraries Used:
- PyTorch
- Alpaca
- Discord
- Sklearn

## Install libraries
```powershell
pip install -r requirements.txt
```
